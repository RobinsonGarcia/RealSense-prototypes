""""Prototype""""

Medir area usando imagens adquiridas com a camera RGB-D:

execute prototype-measure-area.exe e selecione o arquivo odb adequado

clique com o botao esquerdo no ponto inicial e mantenha o botao pressionado até o botao final. O valor 
da distancia sera exibido em vermelho uma vez que o botao esquerdo tenha sido liberado.

aperte "c" para concluir o programa 

aperte "r" para apagar os contornos existentes e recomeçar

OBS: RESULTADOS NAN INDICAM QUE A IMAGEM DE PROFUNDIDADE ESTA COM UMA QUALIDADE RUIM
