Instruçoes!

O programa cria automaticamente a pasta "recorded_bag_videos" para armazenar dados. \
Durante a execuçao o usuario controla o programa atraves do teclado e o mouse.

******* Modo de gravaçao *******
Uso:
------
Keyboard: 
    [p]     Pausa a gravaçao e inicia o MODO de MEDICAO
    [g]     Exporta nuvem de pontos para arquivo ply (./out.ply)
    [q]     Quit / encerra o modo de gravaçao


******* Modo de mediçao (distancias) *******
Uso:
------
Mouse: 
    Aperte o botao esquerdo do mouse no ponto de inicio e mantenha pressionado ate
    o ponto de termino. A medicao sera indicada na tela.

Teclado: 
    [r]     Reinicia visualizaçao / apaga medicoes feitas anteriormente
    [s]     Salva imagem (com mediçoes) para PNG (./out.png)
    [q]     Quit / encerra o modo de mediçao
